# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-07

### Added
- Initial public release of SpanishFly suite.
- Persona module with local-first character generation workflow.
- One-click installers for suite and Persona.
- Automatic environment bootstrap without requiring preinstalled Python.
- Guided system requirement checks with OK/WARN/ERROR summary.
- Optional Hugging Face model download flow during installation.
- Marketing-oriented README with usage guide for non-technical users.

### Security
- Local credential storage guidance for Hugging Face tokens.
- .gitignore rules to avoid committing credentials, outputs, and local environments.
